# 模板语法
vue 多数情况使用的是基于 HTML 的模板语法, 也可以用 jsx
模板语法中，Vue 将模板编译成虚拟 DOM 渲染函数, 允许开发者以声明式的方式将 DOM 和底层组件实例的数据绑定在一起；

## mustache
双大括号 (大胡子)
Mustache 中可以是任意文本, data 的属性或 js 表达式
