# 补充
## link 元素
``` html
<link rel="stylesheet" href=./a.css">
```
rel：指定链接类型 , eg: icon  ,stylesheet
[常见的链接类型](https://developer.mozilla.org/zh-CN/docs/Web/HTML/Link_types)
href：此属性指定被链接资源的 URL。 URL 可以是绝对的，也可以是相对的。

## CSS 颜色的表示方法
- 颜色关键字
- rgb
```css
 color :rgb(255, 255, 255);

 color:rgba(0, 0, 0, .7);

 color: #00000070;

```
