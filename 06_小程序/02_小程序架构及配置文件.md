# 双线程模型
- wxml 和 exss 运行于渲染层，使用 WebView 线程渲染（一个程序有多个页面，会使用多个 WebView 的线程）。
- js 脚本运行于逻辑层，使用 JsCore 运行 JS 脚本。
- 这两个线程都会经由微信客户端（Native）进行中转交互。

```json
  "rules": [{
  "action": "allow",//配置页面能否被搜索到
  "page": "*"
  }]
```
