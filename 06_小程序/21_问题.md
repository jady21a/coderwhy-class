# 跳转页面
app.json 必须注册
父页面不能加后缀名
```js
// index.js
Page({
  data:{
    pages:[
      { name:"aa",path:'/pages/aa/aa' }//aa后面不能加后缀
    ]
  },
  onBtnClick(event) {
    const item = event.target.dataset.item
    wx.navigateTo({
      url: item.path,
    })
  }
})
```

```html
<!--index.wxml-->
<view>
  <block wx:for="{{pages}}" wx:key="name">
    <button type="primary" bindtap="onBtnClick" data-item="{{item}}">
      {{item.name}}
    </button>
  </block>
</view>
```
